# Ficha Express: ZERO-SHOT

---

## Que es
Pedirle a la IA que haga algo SIN darle ejemplos. Confiar en que ya sabe del tema.

## Cuando usarla
- Tareas simples y directas
- El 80% de las veces

## Formula
```
[ROL] + [TAREA] + [CONTEXTO BREVE] + [FORMATO]
```

---

## Ejemplo

```
MAL:
Hablame de futbol

BIEN:
"Eres un comentarista de ESPN. Explicame que es el fuera de lugar en el futbol
usando una analogia con Minecraft. Maximo 50 palabras."
```

## Recuerda
Zero-Shot funciona MUY bien... siempre que NO seas vago. El secreto es ser especifico desde el inicio.
