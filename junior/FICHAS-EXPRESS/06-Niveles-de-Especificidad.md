# Ficha Express: LOS 5 NIVELES DE ESPECIFICIDAD

---

## De vago a quirurgico (ejemplo con videojuegos)

| Nivel | Prompt |
|---|---|
| **1. Vago** | "Dime algo de juegos" |
| **2. Basico** | "Recomiendame un juego" |
| **3. Dirigido** | "Recomiendame un juego de accion para Xbox" |
| **4. Detallado** | "Recomiendame un juego de accion para Xbox, multijugador, presupuesto $30" |
| **5. Quirurgico** | Agrega: rol especifico, edad, gustos, tiempo disponible, que tipo de respuesta quieres... |

---

## La diferencia es enorme

- Nivel 1: Respuesta inutil
- Nivel 3: Respuesta aceptable
- Nivel 5: Respuesta PROFESIONAL lista para usar

## Truco mental
"Si fuera un experto humano, que informacion necesitaria para hacer esto perfectamente?" Lo que respondas, INCLUYELO.
