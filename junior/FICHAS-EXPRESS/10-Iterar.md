# Ficha Express: ITERAR

---

## El secreto mejor guardado
NADIE escribe el prompt perfecto al primer intento. Los profesionales iteran.

---

## El ciclo de iteracion

```
ESCRIBIR -> PROBAR -> EVALUAR -> AJUSTAR -> VOLVER A PROBAR
   ^                                                  |
   |__________________________________________________|
```

---

## Ejemplo real

**Intento 1:**
```
"Escribe un cuento"
```
Resultado: generico y aburrido.

**Intento 2 (agrego tema):**
```
"Escribe un cuento de un dragon que no sabe volar"
```
Resultado: mejor, pero infantil.

**Intento 3 (agrego tono):**
```
"Escribe un cuento de un dragon adolescente que no sabe volar, en tono
divertido como un episodio de serie animada."
```
Resultado: mas cerca de lo que queria.

**Intento 4 (especifico TODO):**
```
"Eres un guionista de Pixar. Escribe un micro-cuento de 200 palabras sobre
un dragon de 15 anos que no puede volar. Sus amigos se burlan. Un dia descubre
que puede hacerse invisible. Incluye un dialogo comico con su amigo murcielago."
```
Resultado: EXACTAMENTE lo que buscaba.

---

## Recuerda
El prompt 1 es un borrador. El prompt 3 es oro. No te rindas en el intento 1.
