# Ficha Express: FEW-SHOT

---

## Que es
Darle a la IA 2-3 EJEMPLOS de lo que quieres ANTES de pedirle la tarea.

## Cuando usarla
- Necesitas un formato o estilo MUY especifico
- Zero-Shot no logra el resultado que buscas

## Formula
```
[INSTRUCCION] + [EJEMPLO 1] + [EJEMPLO 2] + [TAREA NUEVA]
```

---

## Ejemplo

```
Convierte frases a estilo TikTok (5 palabras max):

"Me fue mal en el examen" -> "RIP mi promedio 😭 F"
"Gane en Fortnite" -> "Crown royale facil 🏆"

Ahora convierte: "Manana no hay clases"
```

## Cuidado
TODOS los ejemplos deben seguir el MISMO patron. Si cada ejemplo usa un estilo distinto, la IA se confunde.
